# Vue指令

## v-cloak 

1. 和 css ·[v-cloak] { display: none }· 一起用，可以隐藏未编译的 Mustache 标签直到实例准备完毕
	- 避免页面加载时出现闪烁的问题


## v-once只渲染一次

1. 只渲染组件和元素一次，随后的重新渲染，元素和组件及其所有的节点都会被看做是：静态页面并跳过
	- 用于优化性能，创建低开销的静态组件

2. 如果组件包含大量的静态内容，在根组件上添加 `v-once`
	- 让内容只计算一次，然后缓存起来

```js
<Card v-once>
	<h2>{{title}}</h2>
</Card>
```
